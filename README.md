# Weather App 🌦️

A simple weather application that shows real-time weather information using an API.

---

## 🚀 Features
- Search weather by city name
- Shows temperature, humidity, and weather condition
- Real-time API data
- Simple and responsive UI

---

## 🛠️ Tech Stack
- HTML / CSS 
- Python/Django
- Weather API (OpenWeatherMap / etc.)
